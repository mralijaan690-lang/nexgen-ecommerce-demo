# NexGen E-Commerce Demo

A modern, responsive e-commerce demo showcasing a next-generation online shopping experience with product cards, hero sections, signup/login forms, and clean UI built with HTML, CSS, and Tailwind.

## Features
- Responsive hero section with CTA button
- Product cards with images, descriptions, and ratings
- Signup/Login forms for user authentication
- Modern UI design with Tailwind CSS
- Ready for GitHub Pages deployment
